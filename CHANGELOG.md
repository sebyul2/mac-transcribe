# 변경사항 노트

## v0.2.0

- 실시간 번역 (Live Translation) 재도입
  - 검증된 스트리밍 번역 방식으로 처음부터 재구현: 말하는 중인 발화 전체를 매번 새로 번역하고(re-translation), 연속된 두 번역 결과가 일치하는 앞부분만 흰색으로 확정(local agreement) — 한번 흰색이 된 단어는 다시 바뀌지 않음
  - 아직 움직이는 나머지 부분은 회색으로 표시, 원문은 자막에 표시하지 않음
  - 이미 화면을 지나간 발화는 최신 것만 뒤늦게 번역하고 더 오래된 것은 건너뜀 — 현재 발화의 번역이 항상 우선
  - 메뉴에 Live Translation 토글과 Translation Language 선택(English / 한국어 / 日本語 / 简体中文) 복귀
  - 통역 세션은 원문 대화만 저장 (회의록 미생성)

- 자막이 끊기지 않고 이어짐
  - 하단 자막이 최대 3줄까지 스크롤 방식으로 표시 — 완성된 줄은 위로 밀려 올라갈 뿐 읽는 중에 사라지지 않음
  - 번역이 아직 안 됐거나 실패했을 때(사용 한도·오프라인 등)는 원문을 회색으로 띄우고, 번역이 도착하면 그 자리에서 흰색으로 교체 — LLM이 막혀도 기본 전사 자막은 항상 보임

- 발화 구분 표시 개선
  - 발화 사이 구분이 항상 줄바꿈 한 번으로 통일 — 전사와 자막에 빈 줄이 더 이상 생기지 않음

## v0.1.0

- 동시통역 모드 (Live Translation)
  - 긴 발화 녹음에서 Live Translation을 켜면 실시간 번역 자막이 표시됨
  - 2단 하이브리드 번역: 온디바이스 Apple Translation이 ~100ms에 초벌(회색)을 깔고, LLM 번역이 도착하면 흰색으로 교체
  - 번역 대상 언어 선택 (English / 한국어 / 日本語 / 简体中文)
  - 세션 종료 시 원문 대화만 저장 (통역 세션은 회의록 미생성)

- 발화(화자) 단위 분리
  - 인식기의 단어별 오디오 타임스탬프에 숨은 쉼을 감지해 발화 단위로 문장을 끊음 — 배경음악이 있는 영상에서도 동작
  - 발화가 바뀌면 전사에 줄바꿈, 번역 컨텍스트도 발화 경계를 넘지 않음
  - 무음 기반 보조 감지(조용한 소스) + 60자 초과 시 강제 확정(쉼 없는 연속 발화 대비)

- 시스템 오디오 입력
  - 마이크 대신 컴퓨터에서 나는 소리(통화, 영상)를 입력으로 선택 가능 (메뉴 → Audio Source)
  - Screen Recording 권한 필요 — Permissions 창에 항목 추가

- 자동 회의록 (기본 꺼짐)
  - 녹음 종료 후 LLM이 회의록(참석자/논의/결정/액션 아이템, 필요 시 Mermaid 다이어그램)을 Markdown으로 생성
  - 용어집 반영

- 자막 개선
  - 통역 모드 3줄 표시 (일반 녹음 2줄), 초벌 회색 / LLM 확정 흰색 구분
  - 원문(외국어)은 자막에 표시하지 않음

- 키 설정 화면 (메뉴 → Trigger Keys…)
  - 받아쓰기/긴 발화 트리거 자유 재지정 (⌘⇧R 같은 조합키 통째 등록 가능)
  - Karabiner-Elements 환경 지원

- 번역/보정 속도 개선
  - LLM 요청의 reasoning 제거, SSE 스트리밍 표시, HTTP/2 커넥션 재사용, 세션 시작 시 워밍업

## v0.0.3

- 짧은 녹음 모드 단축키 변경
  - Fn -> Ctrl+Fn 으로 변경
  - Fn 단독은 더 이상 녹음을 시작하지 않아, Fn+F키/화살표 조합 시 오발동이 사라짐

- 긴 발화 녹음 모드 추가 (macOS 26 이상 지원)
  - Ctrl+Shift+Fn으로 시작/종료하는 녹음 모드
  - 결과는 파일로 저장됨: `~/Documents/MacTranscribe/transcript-날짜.txt`
  - macOS 26의 **장시간 전사 엔진(SpeechAnalyzer)** 을 사용하여, 긴 무음이 있어도 안정적으로 동작
  - 유실 방지: 원본 오디오 백업(`recording-날짜.m4a`) + 부분 전사 2초마다 자동 저장
  - 실시간 자막 오버레이(화면 하단, 최대 2줄, 마우스오버 시 닫기)와 전사 창(메뉴 → Transcript Window…) 제공
  - 녹음 중에는 메뉴바 아이콘이 ⏺ 로 표시됨

- LLM Refinement에서 ChatGPT OAuth 모드 지원
  - API 키뿐만 아니라 ChatGPT 계정으로 로그인 가능 (클로드는 정책상 지원이 어렵습니다..)
  - 모델: gpt-5.4-mini(기본) / gpt-5.5 / gpt-5.4
  - 긴 발화도 저장 후 백그라운드에서 보정 (원문 먼저 저장, 실패해도 원문 보존)

- 용어집 추가
  - LLM Settings → Glossary에서 텍스트 파일 첨부/편집
  - 한 줄에 용어 하나, `잘못된표기 -> 올바른표기` 매핑 지원
  - 음성 인식 힌트와 LLM 보정 프롬프트 양쪽에 반영

- 버그 수정
  - 붙여넣기 시 이전 클립보드 내용이 들어가던 레이스 수정
  - 긴 무음 후 받아쓰기 인식기가 먹통 되던 문제에 자동 복구 추가
  - 녹음 시작/종료 연타 시 앱이 강제 종료되던 크래시 수정
